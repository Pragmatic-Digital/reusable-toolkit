# @pragmatic/ui

Reusable React UI components for Pragmatic Digital projects.

## Installation

```bash
npm install @pragmatic/ui
```

## Usage

```tsx
import { /* your components */ } from '@pragmatic/ui';
```

## Development

See the main repository README for development instructions.

## Storybook

To view components in Storybook, run from the root directory:

```bash
pnpm storybook
```
